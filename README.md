# Betax Viewer

Video altyazılarıyla kelime öğrenmek için hazırlanmış bir görüntüleme aracı.

Betax Studio'da oluşturulan kelime kartlarını ve video altyazılarını görüntüler.

## Özellikler

- 🎬 YouTube videoları ile senkronize altyazı
- 🎯 Tıklanabilir kelimeler ve detay görüntüleme
- 📚 Sözcük listesi görünümü
- 🌙 Karanlık tema

## Geliştirme

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
vercel --prod
```
